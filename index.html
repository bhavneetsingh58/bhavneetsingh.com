Project : Log Printer

Binary Seach Based Date Range Log Printer Script

Binary Seach Based Date Range Log Printer Script

Accepts:-

-->Start Date (Date to Start printing logs from ) ,

-->End Date (Date to Start printing logs from ) ,

-->LogFile

Uses Binary search to find the First occurance of Start Date and Last Occurance of End Date & then prints desired Logs to console

Exported Exe and Code available @ github.com/bhavneetsingh58/Log_Date_finder
